Abstract of the research

JavaScript is one of the languages that use web pages and is currently being developed by Netscape and Mozilla JavaScript was initially intended for amateurs and non-programmers, but it increased interest in it and attracted the attention of professional programmers after adding new technologies such as the spread of AJAX technology that was to speed in the interaction between the server and the client. JavaScript uses interactive web pages, and to provide web applications including games; It is used by most websites and supported by almost all browsers without the need for external plugins.
